# Data Description

## Data Sources
- KIPRIS: fintech-related patent applications (city-level)
- KOSIS: GRDP and city-level macro controls
- Other sources : internet penetration, industry structure, etc.

## Folder Structure
- `raw/`: raw source data 
- `processed/`: cleaned datasets used in regression analysis

## Key Files (Processed)
- `panel_final_Fintech_national.csv`: final city-year panel dataset for baseline regressions
- `variable_dictionary.xlsx`: variable definitions and construction notes

## Reproducibility Notes
If raw data cannot be shared due to licensing restrictions, this repository provides:
- processed analysis-ready datasets
